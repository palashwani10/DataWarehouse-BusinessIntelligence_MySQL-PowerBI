# DataWarehouse-BusinessIntelligence_MySQL-PowerBI
 ● End-to-end DWBI project using a star-schema data warehouse. Generated 19,958,064 synthetic records with Python Faker, performed data cleansing and ETL, and built interactive Power BI dashboards. Includes exploratory analysis, forecasting, and predictive analytics for business insights.
